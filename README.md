# Luna 2.0 Meet Companion 🤖🎤

Luna is a virtual Google Meet companion that joins calls, listens to verbal conversations via captions, and replies using a synthetic voice generated by Gemini.

> [!NOTE]
> This application runs completely independently. Other users do **not** need Jetski or any special AI development environments to run it. They only need standard runtimes (Node.js, Python, and Chrome) installed on their computer.

---

## 📋 Prerequisites
Before running Luna, ensure your computer has:
1. **Node.js** (v18+)
2. **Python 3** (v3.10+)
3. **Google Chrome**
4. **Google Cloud SDK (gcloud)**: Required to authenticate the Vertex AI Gemini model.
   - Install from [cloud.google.com/sdk](https://cloud.google.com/sdk)
   - Run `gcloud auth application-default login` to set up your credentials.

---

## 🚀 Getting Started

### Step 1: Automatic Environment Setup
Open your terminal inside the unzipped folder and run the setup script:
```bash
./setup.sh
```
This script will automatically verify your runtimes and install the required Python dependencies (`websockets`).

### Step 2: Configure your API Key (External Users Only)
If you do **not** have a corporate Google Cloud account:
1. Make a copy of `.env.template` and rename it to **`.env`** in the same folder.
2. Open the `.env` file in any text editor and paste your free Gemini API Key (generated from [aistudio.google.com](https://aistudio.google.com/)).

### Step 3: Running Luna

#### Option A: Quick Double-Click (macOS)
Simply double-click the **`Start_Luna.command`** file inside the folder. It will:
- Spin up the local server.
- Open your default web browser to the dashboard portal (`http://localhost:8000`).

#### Option B: Manual Start (macOS, Windows, or Linux)
Run the Node.js server manually from your terminal:
```bash
node server.js
```
Then, open your web browser and navigate to **`http://localhost:8000`**.

---

## 🛠️ How It Works
- **Node.js Server (`server.js`)**: Serves the control portal webpage, coordinates browser events, and serves TTS audio.
- **Gemini Brain (`luna_brain.py`)**: Connects to Vertex AI using your local Google Cloud application-default credentials to generate responses.
- **Chrome Bot (`run_luna_meet.sh`)**: Opens a headless Chrome instance that joins the call, runs the extension to read speech captions, and streams voice audio back.
